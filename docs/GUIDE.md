# DisciplineOS — User Guide

DisciplineOS is your **Personal Recovery Operating System** — habits, daily routines, medicines, recovery timelines, emergency SOS, journal, and trigger intelligence.

**Live app:** https://shamikhahmed.github.io/SteadyCap/

---

## First launch

1. Open the live URL in **Safari** on iPhone.
2. Complete onboarding — set habits, quit dates, and profile.
3. Land on **Today** — your home checklist and recovery pulse.
4. Install as PWA: **Share → Add to Home Screen**.
5. Optional: enable **Push Reminders** in Profile for medicine alerts (iOS 16.4+).

---

## Navigation

| Tab | Purpose |
|-----|---------|
| **Today** | Routines checklist, habit progress, one-tap SOS, gentle nudges |
| **Recovery** | Hourly + daily body-healing timelines, craving forecast |
| **SOS** | Emergency craving protocol, grounding tools |
| **Learn** | Recovery knowledge library |
| **Journal** | Daily entries, mood, triggers |
| **You** | Medicines, skincare/hair routines, habits, settings, export |

---

## Today screen

Your daily command centre:

- **Today's Routines** — medicines, skincare AM/PM, hair AM/PM, weekly steps in one checklist
- **SOS one-tap** — instant craving protocol without hunting through menus
- **Habit progress** — recovery score, streaks, body-healing milestone preview
- **Gentle nudges** — in-app reminders for missed medicine/routine times

---

## Medicines

Configure in **You (Profile)**:

- Fixed times **or** as-needed scheduling
- Push notifications (with permission) + in-app toasts
- Add, edit, enable/disable any entry

---

## Skincare & hair routines

- AM / PM checklists per category
- Weekly steps for deeper care
- Set once in Profile, check off daily on Today

---

## Habits

- **12 built-in habits** — smoking, vape, alcohol, porn, social media, masturbation, weed, gaming, sugar, caffeine, and more
- **Custom habits** — your own name, icon, and quit date
- Streaks and recovery score update as you log progress

---

## Recovery database

- **Hourly body-healing milestones** for the first 72 hours per habit type
- Daily and weekly milestones beyond 72h
- Full timeline on **Recovery** tab with body system scores
- Craving forecast from journal trigger patterns

---

## SOS

When cravings hit:

1. Tap **SOS** from Today or the bottom nav.
2. Follow the phased protocol: pause, ground, breathe, decide.
3. Use grounding tools without leaving the flow.

Designed for one-thumb use under stress.

---

## Journal

Log mood, triggers, and reflections. Journal data feeds craving forecasts and recovery insights.

---

## Profile & data

- Manage medicines, routines, and habits
- Export / import JSON backup
- Push reminder permissions
- Reset (irreversible)

All data stays in `localStorage` on your device.

---

## Offline & privacy

- Service worker caches app shell after first visit
- No accounts, no servers, no analytics
- Airplane mode works after initial load

---

## Troubleshooting

| Issue | Fix |
|-------|-----|
| Push reminders silent | iOS Settings → Notifications → allow for installed PWA |
| Nav missing | Complete onboarding |
| Timeline empty | Confirm habit quit date and allow 72h milestone seed |
